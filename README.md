# 🎬 GREENLIGHT — A Movie Studio Roguelite

A push-your-luck card game where you're a Hollywood studio head making movies, building your reputation, and surviving the chaos of Tinseltown.

Built as a Vite + React + TypeScript web app.

## Play

```bash
npm install
npm run dev
```

## How It Works

- **5 Seasons** — Pick scripts, cast talent, push your luck in production, hit box office targets
- **Push Your Luck** — Draw production cards one at a time. Good cards add quality, bad cards bring chaos. 3 bad = DISASTER
- **Heat vs Skill** — Volatile talent (high Heat) makes productions wilder but potentially better
- **Studio Perks** — Build your studio identity with persistent upgrades between seasons
- **Escalating Pressure** — Box office targets grow each season. Miss too many and you're fired
